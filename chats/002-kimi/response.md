- Deeper search trees in code generation
- More extensive test-case generation
- Real-time iterative refinement (AI as pair programmer with instant response)
- Monte Carlo Tree Search for architectural decisions

1. **KV cache incompatibility:** Diffusion models use bidirectional attention; standard KV caching assumes autoregressive generation. Requires new memory architectures.

3. **Quality ceiling:** Mercury 2 matches Haiku/Mini tier, not Opus/GPT-5.2 tier. Diffusion may have fundamental quality tradeoffs vs. autoregressive at frontier.
